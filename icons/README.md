# Extension Icons

This directory should contain the extension icons in PNG format:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

For development purposes, you can create simple placeholder icons or use any PNG images with the correct dimensions.

To create placeholder icons quickly, you can use online tools like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

Or use an image editor to create simple colored squares with the extension name/logo.
